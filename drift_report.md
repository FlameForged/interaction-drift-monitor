# Interaction Drift Report

This report was generated from the sample early-session and late-session text files in this repository.

## Overview

This prototype compares an early-session text sample with a late-session text sample to estimate basic lexical similarity and possible interaction drift.

## Early Sample

The early sample represents a general, task-focused AI response with limited personalization and minimal continuity beyond the immediate prompt.

## Late Sample

The late sample represents a more personalized, research-aware AI response that references established interests and connects the immediate prompt to a broader project context.

## Notes

This is a lightweight prototype. It uses word-frequency comparison rather than embeddings or deep NLP methods.

Future versions may use:

- sentence embeddings
- semantic similarity scoring
- longitudinal visualization
- model-response clustering
- drift categories tied to qualitative coding

## Research Context

This report is part of an exploratory project translating qualitative human-AI interaction research into early technical evaluation artifacts.
